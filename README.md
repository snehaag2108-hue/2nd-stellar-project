# 2nd-stellar-project
🔥 Stellar Token Burn Mechanism

A Soroban smart contract project implementing a Token Burn Mechanism on the Stellar blockchain. This project enables controlled token supply reduction to increase scarcity, stabilize value, and improve token economics.

---

📌 Project Description

This project is a smart contract built using Soroban (Stellar’s smart contract platform) that allows users or authorized entities to permanently remove tokens from circulation.

Token burning is a widely used mechanism in modern blockchain systems to:

Reduce total supply

Increase token value (deflationary model)

Control inflation

Enhance long-term sustainability

---

🚀 What It Does

The contract provides functionality to:

🔥 Burn tokens from a user account

📉 Reduce total token supply permanently

🔐 Ensure secure and authorized burn operations

📊 Maintain transparency of burned tokens

---

✨ Features

✅ Secure token burn function

✅ Supply reduction tracking

✅ Authorization checks before burning

✅ Event logging for transparency

✅ Compatible with Soroban token standards

✅ Lightweight and efficient smart contract

---

🧠 How It Works

User calls the burn() function

Contract verifies:

Sufficient balance

Authorization

Tokens are removed from the user’s balance

Total supply is reduced

Burn event is recorded on-chain
